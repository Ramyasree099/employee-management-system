# Employee Management System

A Full Stack Employee Management System built using Spring Boot, MySQL, and HTML/CSS.

## 🚀 Features

- Add Employee
- View All Employees
- Update Employee Details
- Delete Employee
- Responsive and clean UI
- RESTful API integration
- MySQL Database connectivity using JPA (Hibernate)

---

## 🛠 Tech Stack

**Backend:**
- Java
- Spring Boot
- Spring Data JPA
- Hibernate

**Database:**
- MySQL

**Frontend:**
- HTML
- CSS
- JavaScript (Fetch API)

**Version Control:**
- Git & GitHub
